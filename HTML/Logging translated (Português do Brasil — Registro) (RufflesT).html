Mod para Keep Talking and Nobody Explodes Registro

A Respeito do Registro

Este módulo por si só arruina por completo a essência do jogo.

NOTA: Este módulo foi criado para praticar e/ou testar outros módulos, e não deve ser usado em uma bomba séria.

Este módulo não pode causar erros.

  • Este módulo foi criado por um programador ruim. Abuse deste fato causando uma IndexOutOfRangeException ao apertar para voltar ou avançar para fora do limite de páginas.
  • A qualquer momento, veja o registro de output de outro módulo clicando em seu nome. Cause uma IndexOutOfRangeException para voltar para o menu principal.
  • Causar uma IndexOutOfRangeException no menu principal desarma este módulo.
  • Este módulo vai registrar toda vez que o usuário usar este módulo para ler linhas do registro.
